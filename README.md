# Chennai Restaurant Data Analysis

### 📋 Project Overview
This project involves analyzing a restaurant dataset from Chennai to gain insights into customer preferences, cuisine popularity, and pricing trends. The analysis was performed using **Microsoft Excel**.

### 🛠 Tools Used
*   **Microsoft Excel:** Data Cleaning, Pivot Tables, Data Visualization.
*   **Techniques:** Data Cleaning, Exploratory Data Analysis (EDA), Dashboard Creation, Slicers.

### 📊 Key Insights
*   **Cuisine Winner:** South Indian cuisine has the highest average rating.
*   **Budget-Friendly:** Arabian cuisine offers the best quality-to-price ratio.
*   **Market Trend:** A significant number of restaurants are focusing on the budget segment (< ₹300).

### 📁 Files Included
*   `Restaurant_Data_Analysis.xlsx`: Contains Raw Data, Pivot Tables, and the Interactive Dashboard.

### 🚀 How to view
Download the Excel file and enable macros/content to interact with the Slicers and Charts.
